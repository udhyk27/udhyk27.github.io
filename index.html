01
[Flutter] 애니메이션 SVG 적용과 앱 구조 개선하기
Flutter · Android | blog ↗
›
상황
Flutter는 애니메이션 SVG를 기본 지원하지 않고, 무거운 초기화 로직이 main에 몰려 스플래시가 늦게 나타나는 문제가 있었음.
원인
flutter_svg는 정적 SVG만 렌더링 가능. Firebase 초기화·Remote Config 호출이 화면 표시보다 먼저 실행되어 체감 속도 저하.
해결
SVG를 HTML에 삽입 후 WebView 렌더링으로 애니메이션 재생. Future.wait으로 초기화 병렬 처리, API 클래스를 Singleton 패턴으로 개선.
02
[프로그래머스] 택배 상자 꺼내기
Algorithm · Java | blog ↗
›
상황
2차원 배열로 지그재그 구조를 직접 구현했으나, 배열 크기 계산과 층마다 방향 전환 처리가 번거로워 더 나은 방법을 탐색.
원인
배열을 직접 생성하면 불필요한 메모리 사용과 순회가 발생. 수학적으로 위치를 바로 계산할 수 있음을 인식.
해결
상자 번호로 행·열 위치를 직접 계산하는 방식으로 전환. (num-1)/w로 행, (num-1)%w로 열을 구하고 홀수 행은 지그재그 보정 적용.
03
[Flutter] 앱을 살리며 겪은 iOS 트러블슈팅
Flutter · iOS | blog ↗
›
상황
방치된 Flutter 프로젝트 최신화 중 iOS에서만 연속 크래시. SIGABRT → dyld 실패 → EXC_BAD_ACCESS → 흰 화면으로 단계적으로 이어짐.
원인
Flutter 버전 자체 버그, Remote Config URL 빈 문자열 미처리, AppDelegate에서 FlutterViewController가 window에 연결되지 않은 것이 최종 원인.
해결
Flutter 3.38.5 업그레이드 후 null 방어 코드 추가. FlutterViewController → UIWindow → rootViewController 연결을 명시적으로 작성하여 해결.
